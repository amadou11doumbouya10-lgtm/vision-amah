# Vertex Shop

Boutique e-commerce Shopify dédiée aux soins de la peau haut de gamme.

- **Niche** : cosmétiques et sérums de luxe (produit phare : Vitaminic Skin Set, vitamine C)
- **Identité de marque** : « Glow Different » — palette noir / or / orange, typographie élégante (Cormorant Garamond pour les titres)

## Stack

- Shopify, thème Liquid personnalisé
- HTML/CSS sur-mesure pour les sections du thème

## Périmètre de l'intervention

- SEO de la page d'accueil (titre, méta-description)
- Page produit : description enrichie, bénéfices en bullet points, ingrédients actifs, mode d'emploi
- Habillage visuel : palette dédiée, bouton d'achat, bannière promotionnelle
- Page d'accueil : section hero, argumentaire « pourquoi nous choisir », témoignages, produits mis en avant
- Performance : optimisation des images, balises Open Graph pour les réseaux sociaux

## Notes

- Aucun token d'accès à la boutique n'est documenté ici.
